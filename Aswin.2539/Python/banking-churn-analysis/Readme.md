# 📊 Banking Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer data to identify patterns and factors influencing customer churn in a banking environment. The goal is to derive actionable insights that can help improve customer retention strategies.

---

## 🎯 Objectives
- Identify key factors contributing to customer churn  
- Perform data cleaning and preprocessing  
- Conduct exploratory data analysis (EDA)  
- Generate insights for business decision-making  

---

## 📂 Dataset
- Dataset contains **10,000+ customer records**
- Features include:
  - Age
  - Balance
  - Tenure
  - Geography
  - Activity status
  - Churn (target)

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## ⚙️ Steps Performed

### 1. Data Cleaning
- Handled missing values  
- Removed inconsistencies  
- Converted data types  

### 2. Data Transformation
- Encoded categorical variables  
- Created new features for analysis  

### 3. Exploratory Data Analysis (EDA)
- Analyzed churn distribution  
- Studied relationships between features and churn  
- Identified trends using visualizations  

---

## 📊 Key Insights

- Customers with **low balance** are more likely to churn  
- Customers with **low activity levels** show higher churn rates  
- Customers with **short tenure** tend to leave earlier  
- Certain geographic regions show higher churn patterns  

---

## 📈 Visualizations
- Churn distribution charts  
- Balance vs Churn analysis  
- Tenure vs Churn trends  
- Activity level comparison  

---

## 💡 Business Impact
- Helps banks identify high-risk customers  
- Supports targeted retention strategies  
- Improves customer engagement and revenue retention  

---

## 🚀 Conclusion
The analysis highlights critical factors affecting customer churn and provides actionable insights that can be used by businesses to improve customer retention and decision-making.
